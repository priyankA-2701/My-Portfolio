## 🚀 Featured Projects

### 🔬 [Automated Melanisation & Encapsulation Detection](https://github.com/priyankA-2701)

> **Real research. Real deployment. Real impact.**

Built at CNR Naples — an end-to-end AI pipeline for immune response analysis in larval microscopy images. From raw JSON annotations to a deployed Streamlit app used by scientists across 3 countries.

- 🏆 **Attention U-Net:** IoU `0.81` · Dice `0.89`
- 📊 **Biological accuracy:** MI Pearson correlation `0.94`
- 🌍 **Multi-country:** Italy 🇮🇹 · Germany 🇩🇪 · Croatia 🇭🇷
- 🚀 **Deployed:** Streamlit real-time inference app
- 🔍 **Insight:** Diagnosed Italy domain shift via LoCo cross-validation

`PyTorch` `OpenCV` `Albumentations` `SciPy` `Streamlit` `Plotly`

---

### 🚀 [SpaceX Falcon 9 — Launch Outcome Prediction](https://github.com/priyankA-2701)

> **Complete ML pipeline: API → model → interactive dashboard.**

Predicted first-stage landing success using 5 classifiers, GridSearchCV tuning, and a fully interactive Plotly/Dash dashboard.

- 🔗 Data via **REST API** + **web scraping**
- 🤖 Compared: `Logistic Regression` `SVM` `Decision Tree` `Random Forest` `KNN`
- 📈 Interactive dashboard: filter by site, orbit, payload
- 💡 Key finding: payload mass + launch site drive success probability

`Scikit-learn` `Pandas` `Plotly` `Dash` `BeautifulSoup`

---

### ⚽ [European Football & FIFA Data Mining](https://github.com/priyankA-2701/Data-Mining)

> **Sports data → scientific insights → 96% prediction accuracy.**

Full data mining workflow: EDA, hypothesis testing, clustering, association rules, and deep learning.

- 🎯 Player rating prediction: **~96% accuracy** (PyTorch + Keras)
- 🧩 K-Means: revealed 4 distinct player archetypes
- 📊 Home advantage: **statistically validated** across thousands of matches
- 🔗 Association rules: meaningful player-team patterns discovered

`PyTorch` `Keras` `Scikit-learn` `Pandas` `Seaborn`
